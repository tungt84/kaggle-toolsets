# kaggle-toolsets

Utilities for day-to-day Kaggle workflows.

## Features

- Python package installable with `pip`
- CLI command: `kaggle-toolsets`
- Built-in setup check command for Kaggle auth

## Requirements

- Python 3.10+
- pip

## Installation

### 1. Clone repository

```bash
git clone https://github.com/tungt84/kaggle-toolsets
cd kaggle-toolsets
```

### 2. Install dependencies and package

Install in editable mode (recommended for development):

```bash
pip install -e .
```

Or install as a normal package:

```bash
pip install .
```
